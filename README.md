A simple and lightweight **Wi-Fi Intrusion Detection System** built using Python and Npcap.  
This tool analyzes nearby Wi-Fi activity, monitors packets, and detects suspicious behaviors such as:

- Deauthentication attacks  
- Unknown MAC addresses  
- Rapid probe requests  
- Packet floods  
- Sudden disconnect patterns  
- Abnormal wireless traffic

This project was developed as part of a cybersecurity learning initiative and is intended for educational and defensive purposes only.
